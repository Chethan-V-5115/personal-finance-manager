# Personal Finance Manager

A modern, frontend-only Single Page Application (SPA) designed to help individuals track their income, expenses, and savings goals in one place. Built with performance and privacy in mind.

## 🚀 Technologies Used
- **React 19** - UI component framework
- **Vite** - Build tool and development server
- **TypeScript** - For type-safe development
- **Tailwind CSS** - For responsive, utility-first styling

## ⚡ Features
- **Dashboard:** Real-time visualization of your total balance.
- **Transaction Tracker:** Add, view, and manage income and expenses.
- **Responsive Design:** Fully optimized for both desktop and mobile screens.
- **Privacy-Focused:** All data is stored locally in your browser—no backend required.

## 🛠️ How to Run Locally
1. Clone this repository:
   ```bash
   git clone https://github.com/Chethan-V-5115/personal-finance-manager.git
   ```
2. Navigate to the folder:
   ```bash
   cd personal-finance-manager
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
4. Start the development server:
   ```bash
   npm run dev
   ```
